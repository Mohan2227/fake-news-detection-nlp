# Fake News Detection using NLP

## Overview
This project uses Natural Language Processing (NLP) and Machine Learning to classify news as fake or real.

##  Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer

##  Features
- Data preprocessing
- Text vectorization using TF-IDF
- Classification using Logistic Regression

## Output
- Predicts whether a news article is fake or real
- Achieves high accuracy

##  How it Works
- Combined fake and real datasets
- Converted text into numerical features
- Trained ML model to classify news

##  Conclusion
This project helps detect misinformation using machine learning techniques.
